\doxysubsection{Detailed Description}
This class wraps the \mbox{\hyperlink{class_core_window_frame}{Core\+Window\+Frame}} class and is responsible for delegating most of the method calls to that class. It is also the top root of the containment hierarchy and is the first component that should be created in your application. All the components that are to be displayed within the window should be added via the \mbox{\hyperlink{class_component_a1004baa17f2e5d6b9e808e908f2a3825}{Component\+::\+Add}} function which this class inherits.
