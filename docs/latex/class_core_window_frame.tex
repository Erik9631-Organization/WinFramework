\doxysubsection{Detailed Description}
The core frame, the raw root of the entire system. The class is wrapped by \mbox{\hyperlink{class_window_frame}{Window\+Frame}} class. This class is responsible for handling the windows messaging, creating events and responsible for the rendering system.
