Detailed Description

This class wraps the CoreWindowFrame class and is responsible for delegating most of the method calls to that class. It is also the top root of the containment hierarchy and is the first component that should be created in your application. All the components that are to be displayed within the window should be added via the Component::Add function which this class inherits.
